# Codex Deep-Work Playbooks

These files are scoped workflow specs for deeper, focused tasks.

They are useful in two ways:

1. Ask Codex to read and follow one directly when you want a focused workflow
2. Use them as specifications if your Codex environment supports explicit delegation and the user
   asks for parallel or delegated work

They are not auto-registered repo-local agents.

## Playbooks

| File | Purpose |
|------|---------|
| `context-loader.md` | Build a briefing on a person, project, incident, or concept |
| `cross-linker.md` | Find missing links and backlink gaps |
| `vault-librarian.md` | Audit note hygiene and structural health |
| `brag-spotter.md` | Find uncaptured wins and competency evidence |
| `review-prep.md` | Aggregate evidence for a review period |
| `review-fact-checker.md` | Verify review claims against vault sources |
| `people-profiler.md` | Create or update person notes from source material |
| `slack-archaeologist.md` | Reconstruct events from Slack exports or transcripts |
| `vault-migrator.md` | Classify and migrate notes from another vault |
